# Kvoxy - Мессенджер для разработчиков

Полнофункциональный мессенджер для программистов с фильтрацией по языкам, кастомизацией тем и уникальными функциями.

## Установка и запуск

### Важно: используйте HTTP, не HTTPS!

Браузер может пытаться открыть `https://localhost:8000`, но нужно использовать `http://localhost:8000`

### Способ 1: Автоматический запуск (оба сервера)

```bash
./start.sh
```

### Способ 2: Ручной запуск (рекомендуется)

**Терминал 1 - Backend:**
```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

Backend запустится на `http://localhost:5000`

**Терминал 2 - Frontend:**
```bash
python3 -m http.server 8000
```

Frontend запустится на `http://localhost:8000`

### Откройте в браузере:

**http://localhost:8000** (не https://!)

Если браузер автоматически переключается на HTTPS:
1. Введите в адресной строке: `http://localhost:8000`
2. Или очистите кеш браузера
3. Или используйте режим инкогнито

## Функции

- ✅ Регистрация и авторизация
- ✅ Фильтрация постов по языку программирования
- ✅ Создание постов с файлами (изображения, видео, код)
- ✅ Комментарии с вложенными ответами
- ✅ Система тем (6 тем)
- ✅ Настройки профиля
- ✅ Чаты и каналы
- ✅ Удаление файлов при наведении
- ✅ Подсветка файлов при наведении
- ✅ **Live Code Preview** - уникальная функция для просмотра кода

## Уникальные функции

1. **Фильтрация по языкам** - видишь только посты на твоем языке программирования
2. **Кастомизация тем** - как на Monkeytype, множество тем на выбор
3. **Умная работа с файлами** - подсветка и легкое удаление
4. **Live Code Preview** - предпросмотр HTML/JS/CSS кода в реальном времени
5. **Минималистичный дизайн** - только нужные функции

## API Endpoints

- `POST /api/register` - Регистрация
- `POST /api/login` - Вход
- `GET /api/user` - Получить данные пользователя
- `POST /api/user/update` - Обновить профиль
- `GET /api/posts` - Получить посты (фильтруются по языку)
- `POST /api/posts` - Создать пост
- `POST /api/posts/<id>/comments` - Добавить комментарий
- `POST /api/posts/<id>/like` - Лайкнуть пост
- `POST /api/upload` - Загрузить файл
- `GET /api/channels` - Получить каналы
- `GET /api/themes` - Получить темы
- `GET /api/settings` - Получить настройки
- `POST /api/settings` - Сохранить настройки

## Решение проблем

### SSL ошибка
Если видите ошибку SSL_ERROR_RX_RECORD_TOO_LONG:
- Убедитесь, что используете `http://` а не `https://`
- Очистите кеш браузера
- Попробуйте режим инкогнито

### Backend не запускается
- Убедитесь, что Python 3 установлен
- Проверьте, что порт 5000 свободен
- Установите зависимости: `pip install -r requirements.txt`

### Frontend не загружается
- Убедитесь, что backend запущен на порту 5000
- Проверьте консоль браузера на ошибки
- Убедитесь, что используете `http://localhost:8000`
