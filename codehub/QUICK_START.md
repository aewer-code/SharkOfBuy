# Быстрый старт CodeHub

## ⚠️ ВАЖНО: Используйте HTTP, не HTTPS!

Браузер может автоматически пытаться открыть HTTPS, но сервер работает по HTTP.

## Правильный способ открыть сайт:

1. **В адресной строке браузера введите:**
   ```
   http://localhost:8000
   ```
   или
   ```
   http://127.0.0.1:8000
   ```

2. **Если видите ошибку SSL:**
   - Убедитесь, что используете `http://` (не `https://`)
   - Очистите кеш браузера (Ctrl+Shift+Delete)
   - Используйте режим инкогнито (Ctrl+Shift+N)
   - Попробуйте другой браузер

## Запуск серверов:

### Вариант 1: Один скрипт
```bash
./start.sh
```

### Вариант 2: Отдельные терминалы

**Терминал 1:**
```bash
./start-backend.sh
```

**Терминал 2:**
```bash
./start-frontend.sh
```

## Проверка:

1. Backend должен быть на: `http://localhost:5000`
2. Frontend должен быть на: `http://localhost:8000`
3. Откройте в браузере: `http://localhost:8000`

## Если не работает:

1. Проверьте, что оба сервера запущены
2. Проверьте консоль браузера (F12) на ошибки
3. Убедитесь, что используете `http://` а не `https://`
